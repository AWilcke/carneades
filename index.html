The Carneades model of argumentation¶

This is a Python implementation of the Carneades argumentation model. It closely follows the Haskell Carneades DSL by Bas van Gijzel and Henrik Nilsson, especially this detailed code.

The main module caes.py uses the igraph package for representing dependencies between arguments. For documentation, see

  • igraph tutorial
  • igraph manual and API

Plotting in igraph is dependent on the pycairo library which provides Python bindings to the Cairo library, as illustrated below.
